## 聚龙链（JulongChain）平台
聚龙链平台是一个基于Java的开源联盟链区块链平台。


## 链接
[Jira任务管理系统](http://jira.bcia.net.cn:8082/)

[Gitlab代码库管理系统](http://gitlab.bcia.net.cn:6060/)

## 如何编译
使用IntelliJ IDEA开发环境。

使用maven进行编译。

JDK版本为1.8。

## 当前版本
当前版本为0.8。
